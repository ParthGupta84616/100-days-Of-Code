The festival of lights is approaching and itâ€™s time to send gifts to family and friends. Shivaay is at a store to buy Diwali presents and he has selected a total of n gifts out of all the gifts available at the store. The price of each present is represented in the form of array L of M elements(where L[i] represent the cost of i th gift in Rupees). He initially has X Rupees to spend and he wants to buy as many gifts as possible. Your task is to find the maximum number of gifts, Shivaay can buy with an available purse of X Rupees.

Input Format

The First line will give T, the number of Testcases.
The First line of each testcase will give M( the number of elements in array L) and X, the budget Shivaay has.
The next line of each testcase will give the array L.
Constraints

1<=T<=1000
1<=M<=10,000
1<=X<=1,00,000
1<=L[i]<=1,00,000
Output Format

For Each testcase, output the maximum n possible.

Sample Input 0

1
5 7
1 3 2 4 1
Sample Output 0

4
Explanation 0

Shivaay can buy gifts at indices 0,1,2,4 for a total price of 1 + 3 + 2 + 1 = 7.

def max_gifts(m, x, gifts):
    gifts.sort() 
    count = 0
    total_cost = 0

    for i in range(m):
        total_cost += gifts[i]
        if total_cost <= x:
            count += 1
        else:
            break

    return count

def main():
    t = int(input())
    
    for _ in range(t):
        m, x = map(int, input().split())
        gifts = list(map(int, input().split()))

        result = max_gifts(m, x, gifts)
        print(result)

main()
